    Horário: 21:00min - 22:02min
    Local: Microsoft Teams

    1. Introdução

    A verificação é uma das etapas mais importantes do desenvolvimento de um projeto. Nela os artefatos produzidos são analisados garantindo que os mesmos cumpram com os seus requisitos especificados. Sendo assim, este documento apresenta o planejamento da verificação dos artefatos produzidos pelo Grupo na Etapa 4.
